🚀 NERVANA — PHASE 0 & PHASE 1 PLAN

🌐 PHASE 0: LANDING + VALIDATION (Weeks 1–2)

Goal: Create an initial presence, validate interest, and start building an early user base.

Deliverables

  • Website (Landing Page): Explaining the what, who, and how.
  • Core Messaging: "Daily mental clarity for working professionals."
  • User Capture: A waitlist to collect emails.
  • App Access Funnel: Simple "Download" or "Join" buttons.
  • Basic Trust Layer: Clean design and a clear privacy mention.

Timeline

Week 1: Landing page live

Week 2: Aim for 50–100 waitlist signups

✅ Success Criteria

  • 50–200 people express interest
  • Messaging resonates with target users

📱 PHASE 1: MVP PRODUCT (Weeks 3–8)

Goal: Build a simple, daily-use product that users come back to.

Key Principle: We are not building features — we are building a daily habit.

🧠 What the User Experiences (Daily Journey)

  • 1. Daily Reminder: “Take 1 minute to check in with yourself.”
  • 2. Quick Check-in (30–60 sec):
    • Mood
    • Energy level
    • What’s on your mind (optional)
  • 3. Instant Insight:
    “You seem mentally tired but not emotionally low — likely due to work pressure.”
    👉 User feels understood
  • 4. Small Action:
    • 2-min breathing
    • Write 3 thoughts
    • 10-min focus sprint
    👉 Immediate small improvement
  • 5. Progress Feedback:
    • Daily streak
    • “You showed up today”
    👉 Builds habit consistency

🔁 Core Loop

Check-in → Insight → Action → Feel Better → Return

✨ Expected User Feeling

  • This understands me
  • This is easy
  • I feel slightly better

📦 Deliverables

  • Daily check-in
  • Insight generation
  • Single daily action
  • Streak system
  • Push notifications

❌ Not Included

  • No therapist marketplace
  • No social / leaderboard
  • No complex gamification
  • No heavy AI

Timeline

Week 3–4: Check-in + insight

Week 5–6: Action + streak

Week 7–8: Notifications + polish

📊 Success Criteria

  • 100–300 users try the app
  • 25–35% return after 1 week
  • Users feel it is helpful & easy

🧭 OVERALL STRATEGY

Phase 0 → Validate Interest

Phase 1 → Validate Habit

Focus: Build something users return to daily
