🚀 NERVANA — PHASE 0 & PHASE 1 PLAN

🌐 PHASE 0: LANDING + VALIDATION (Weeks 1–2)

Goal: Create an initial presence, validate interest, and start building an early user base.

Deliverables

  • Website (Landing Page): Explaining the what, who, and how.
  • Core Messaging: "Daily mental clarity for working professionals."
  • User Capture: A waitlist to collect emails.
  • App Access Funnel: Simple "Download" or "Join" buttons.
  • Basic Trust Layer: Clean design and a clear privacy mention.

Timeline

Week 1: Landing page live

Week 2: Aim for 50–100 waitlist signups

✅ Success Criteria

  • 50–200 people express interest.
  • The core messaging resonates with the target audience.

📱 PHASE 1: MVP PRODUCT (Weeks 3–8)

Goal: Build a simple, daily-use product that users come back to.

Core Product Experience

A simple loop: Check-in → Insight → Small Action → Come back tomorrow.

Deliverables

  • Daily Check-in: Fast and simple (under 1 minute).
  • Insight (Clarity): A short, personalized output.
  • Daily Action: One small, doable recommendation.
  • Habit Loop: A daily streak to encourage consistency.
  • Notifications: A gentle daily reminder.

❌ What We Will NOT Build (Yet)

  • No therapist integrations or marketplaces.
  • No social features or leaderboards.
  • No complex rewards or gamification.

Timeline

Week 3–4: Basic app (check-in + insight).

Week 5–6: Add actions and streak tracker.

Week 7–8: Implement notifications and polish the experience.

📊 Success Criteria

  • 100–300 users try the app.
  • 25–35% of users return after one week.
  • Feedback indicates users find it helpful and easy to use.

🧭 OVERALL STRATEGY

Phase 0 → Validate Interest

Phase 1 → Validate Habit

We are building a simple daily mental clarity tool, proving consistent user return before expanding.
